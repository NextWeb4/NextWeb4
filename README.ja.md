## 現在の重点領域

- データ境界が明確な Windows／クロスプラットフォーム向けデスクトップユーティリティ。
- 設定、翻訳、メタデータ、個人作業を支えるオフラインファーストのワークフロー。
- 静的かつ多言語の Web 体験と、管理が必要な場合に限った小さく監査可能なサービス。
- 実際のセットアップ、セキュリティ、パッケージング、保守上の制約を記録する文書。

## 主なプロジェクト

| プロジェクト | 概要 |
| --- | --- |
| [yaml-proxy-editor](https://github.com/NextWeb4/yaml-proxy-editor) | Clash、OpenClash、Mihomo の YAML をローカルで監査・編集する Tauri／React／TypeScript ワークベンチ。 |
| [alias-cockpit](https://github.com/NextWeb4/alias-cockpit) | ローカルのメールエイリアスを生成、マーク、管理する .NET／WinUI 製 Windows ツール。 |
| [deepseek-translation-studio](https://github.com/NextWeb4/deepseek-translation-studio) | Web、API、字幕、キャプチャ、パッケージングに対応する Python／PySide6 翻訳スタジオ。 |
| [NextWeb4.github.io](https://github.com/NextWeb4/NextWeb4.github.io) | 個人 GitHub Pages ブログ、公開プロジェクトの入口、自己記録ジャーナル。 |
