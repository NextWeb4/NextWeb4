## 当前方向

- 为 Windows 与跨平台场景开发数据边界清晰的桌面工具。
- 面向配置、翻译、元数据和个人效率的离线优先工作流。
- 静态、多语言 Web 体验；需要管理能力时只配套小型且可审计的本地服务。
- 在文档中记录真实的安装、安全、打包和维护约束。
